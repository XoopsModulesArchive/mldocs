Personnaliser votre skin
Editer une feuille de style : style*.css
La classe body
body {
  margin : 0px 0px 0px 0px;
  background-color: #FFFFFF;
  font-size: 11px;
  font-family: "Microsoft Sans Serif", Arial, Verdana;
}

Cette classe permet de définir :
La couleur de fond : background-color: #FFFFFF;
cette couleur de fond s'applique à toutes les pages sauf celle du diaporama
la taille du texte par défaut : font-size: 11px;
la police utilisée par défaut: font-family: "Microsoft Sans Serif", Arial, Verdana;
Le premier nom indique la police utilisée par défaut le deuxiéme est utilisée si la premiére n'existe pas etc...

La classe tn
.tn{
  text-align: center;
  border: 1px solid #BBBBBB;
  padding: 0px;
  background-color: #FFFFFF;
}

Cette classe permet de définir :
La couleur de fond des miniatures: background-color: #FFFFFF;
Le contour des miniatures border: 1px solid #BBBBBB;
border : epaisseur de la bordure (0px pour ne pas avoir de bordure)
#BBBBBB : c'est la couleur de la bordure.

La classe bande
.bande {
  font-size: 11px;
  font-weight:none;
  color: #000000;
  background-color: #CCCCCC;
}

Cette classe permet de définir :
La couleur des bande (nom,taille...): background-color: #CCCCCC;
Le style d'écriture des mots de la bande:
font-weight:none; : au lieu de none vous pouvez mettre bold = gras - bolder = très gras - lighter = plus fin.
color: #000000; : couleur de l'écriture

La classe diaporama
Permet de définir la couleur de fond du diaporama.

La classe message
.message {
border: 1px solid #BBBBBB;
padding: 0px;
font-weight:bold;
background-color: white;
}
.message a:link { color:black; text-decoration:none }
.message a:visited { color:black; text-decoration:none }
.message a:hover { color:black; text-decoration:none }


Cette classe permet de définir :
La couleur de fond de la fenêtre du message
La bordure de la fenêtre du message
Le style des liens contenu dans le message :
a:link { color:black; text-decoration:none } : couleur du lien
a:visited { color:black; text-decoration:none } : couleur du lien quand il a été visité
a:hover { color:black; text-decoration:none } : couleur du lien lors du passage de la souris
text-decoration:none : mettez underline à la place de none si vous voulez que le lien soit souligné
