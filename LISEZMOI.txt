Module Proposé
-----------------------------------------------------------------------
mldocs est dessiné en vu de vous apporter une réponse adéquate en matière
de support utilisateurs dans une application "helpdesk" sous la forme
d'un module dans un environnement applicatif XOOPS. 

Pré-requis d'installation
-----------------------------------------------------------------------
XOOPS 2.0.4+

Instructions d'installation
-----------------------------------------------------------------------
Unzippez l'archive fournit dans le répertoire 'modules'. 
Installer l'applicatif en utilisant le panneau de contrôle Xoops usuel. 
Ajustez vos préférences pour le module si cela est nécéssaire.
Ajouter les départements opérateurs des traitements des archives (catégories).
Paramètre les comptes utilisateurs XOOPS en les répartissant dans leur(s) département(s)
Respectifs.
Tout est fin pret!!!

Instructions de mise à jour
-----------------------------------------------------------------------
Unzippez l'archive fournit dans le répertoire 'modules'.
Mettez à jour le module en utilisant le panneau de contrôle Xoops usuel. 
Cliquez sur 'Contrôle de Tables' à partir du menu principal mldocs, ou du menu pop-up.
Contrôlez et Ajustez vos préférences pour le module si cela est nécéssaire.
Tout est fin pret!!!
